## Language Mechanics

### Print

```Java
System.out.print("hi");
```

### Hashmap

```Java
import java.util.*;
Map<Integer, Integer> map = new HashMap<>();
map.put(0, 3);
System.out.print(map.containsKey(0)); // True
System.out.print(map.get(0)); // 3
```

### Math

```Java
int x = 0;
x = Math.max(1,2);
System.out.print(x); // 2
```
